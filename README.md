# 脳活ジグソーパズル（Web版）

高齢者向けの、オフラインで動く無料・広告なしのジグソーパズルPWA。
Vercel（https://nou-jigsaw-web.vercel.app/）で公開し、Farcaster Mini App として配信する純静的サイト。

- 本体コードは Play 版リポジトリ `nou-jigsaw` と重複コピー（変更は両方に反映すること）
- `.well-known/farcaster.json` の accountAssociation は Farcaster Manifest ツールで署名して埋める

アプリ開発：介護と支援の相談どころ「そよぎ」
